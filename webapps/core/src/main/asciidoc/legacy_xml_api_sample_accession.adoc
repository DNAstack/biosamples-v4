== Sample Accession

- It returns all information about a Sample;
- E.g.   link:/biosamples/xml/sample/SAME546131[];


==== Authentication
N/A

==== Arguments
[options="header"]
|===
| Parameter | Notes
|sampleAccession|Required
|groupslist| Optional (Default=false)
|===

==== Example Request
`Get /xml/sample/SAME546131`
